# 更新日志

## v1.0.6

### 新增
- 主题设置中新增“显示访客胶囊”，开启后在首页底部显示访客 IP、地区和运营商信息。
- 主题设置中新增“显示资产卡片”和“资产卡片默认币种”，开启后显示右上角资产统计浮窗、汇率换算、排序、免费资产过滤和交易计算。

### 调整
- 资产卡片的单台服务器价格读取主题合并后的账单数据，会匹配“默认账单货币”和“服务器货币覆盖”配置。

### 修复
- 修复资产卡片在部分缓存或旧账单数据下未按服务器货币覆盖换算，导致 JPY 等价格被按 CNY 汇总的问题。

## v1.0.5

### 新增

- 主题设置中新增「默认账单货币」选项，可统一指定账单价格显示货币。
- 主题设置中新增「服务器货币覆盖」配置，可按服务器 UUID、卡片 ID 或服务器名称单独指定货币。

### 修复

- 修复人民币和日元都显示为 `¥` 时无法区分的问题，账单价格现在会显示为 `CNY 价格` 或 `JPY 价格`。
- 修复按量计费服务器在带货币符号时可能被显示为 `0` 的问题。

### 说明

- 默认不再按服务器地区推断币种；如需覆盖，请在主题管理中配置默认货币或服务器货币覆盖 JSON。
